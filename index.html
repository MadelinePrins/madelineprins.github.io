Naive Bayes Classifier

MACHINE LEARNING

The Naive Bayes Classifier contains two major functions: train() and classify(). The train function takes in a dataset and fills a complex model datastructure with a series of probabilities involving the 'sensitive attribute' and 'classification variable' which will be used during classification. The classify function is run on a separate dataset. Given the trained model it calculates the possible classifications based on the probabilities and appends a new classification column to the dataset.

Modified Bayes Classifer For Binary Sensitive/Classification

MACHING LEARNING

The Modified Bayes Algorithm is built off of the Naive Bayes Classifier foundation. It computes a preliminary baseline discrimination score based on a single iteration of the naive bayes classifier. It then enters a while loop in which it modifies the counts and conditional probabilities for particular classifications and sensitive attributes very slightly to make it more 'fair'. It then reclassifies the data and recomputes the new discrimination score. The loop continues until the discrimination score is minimized to a particular threshold.

Two Bayes Algorithm For Binary Sensitive/Classification

MACHING LEARNING

The Two Bayes Algorithm is built off of the foundations of both the Naive Bayes Classifier and the Modified Bayes algorithm. The distinction comes from a separation of dataFrame objects originally by the binary sensitive attribute in order to train two separate models. The classifier when iterating through the data set then uses and modifies the probabilities from the model corresponding to the particular data instance's sensitive attribute value.
