    • The Naive Bayes Classifier contains two major functions: train() and classify(). The train function takes in a dataset and fills a complex model datastructure with a series of probabilities involving the 'sensitive attribute' and 'classification variable' which will be used during classification. The classify function is run on a separate dataset. Given the trained model it calculates the possible classifications based on the probabilities and appends a new classification column to the dataset.
