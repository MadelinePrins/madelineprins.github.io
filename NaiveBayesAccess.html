Fairness Aware Naive Bayes Classifier

Datastructure Details ** The train() function:
If categorical data type:
 Fill model with “cross attribute probability” for each possible attribute paired with each possible outcome for the class variable

Else if numerical data type: 
 Fill model with conditional mean and standard deviation for a particular attribute and all the possible outcomes for the class variable

** The classify() function:

							variable possibilities
 
 
 

 Else if attribute data type is numerical:

							classification /pre>
