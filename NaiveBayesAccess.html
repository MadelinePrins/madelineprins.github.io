Fairness Aware Naive Bayes Classifier

Datastructure Details The train() function:
If categorical data type:
&emsp Fill model with “cross attribute probability” for each possible attribute paired with each possible outcome for the class variable
Else if numerical data type:
&emsp Fill model with conditional mean and standard deviation for a particular attribute and all the possible outcomes for the class variable
The classify() function:
&emsp For each entry in the dataset:
&emsp &emsp For each of the possible classification:
&emsp &emsp &emsp For each attribute of that entry:
&emsp &emsp &emsp &emsp If the attribute data type is categorical:
&emsp &emsp &emsp &emsp &emsp -calculate probability for each of the possible class variable possibilities
&emsp &emsp &emsp &emsp Else if attribute data type is numerical:
&emsp &emsp &emsp &emsp &emsp -calculate gaussian probability of attribute for each classification
&emsp &emsp &emsp &emsp calculate P(person|classification)*P(classification) / P(person)
&emsp Classify entry based on highest classification probability
