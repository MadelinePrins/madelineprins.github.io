Two Bayes Algorithm For Binary Sensitive/Classification Attributes

  • Download the Code Zip File HERE

    • The Two Bayes Algorithm is built off of the foundations of both the Naive Bayes Classifier and the Modified Bayes algorithm. The distinction comes from a separation of dataFrame objects originally by the binary sensitive attribute in order to train two separate models. The classifier when iterating through the data set then uses and modifies the probabilities from the model corresponding to the particular data instance's sensitive attribute value.
