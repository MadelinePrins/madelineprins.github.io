Modified Bayes Classifier For Binary Sensitive/Classification Attributes

  • Download the Code Zip File HERE

    • The Modified Bayes Algorithm is built off of the Naive Bayes Classifier foundation. It computes a preliminary baseline discrimination score based on a single iteration of the naive bayes classifier. It then enters a while loop in which it modifies the conditional probabilities for particular classifications and sensitive attributes very slightly. It then reclassifies the data and recomputes the new discrimination score. The loop continues until the discrimination score is minimized to a particular threshold.
